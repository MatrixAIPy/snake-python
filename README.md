# Snake Game

A simple Python clone of the classic Snake game, playable directly in the terminal.

**Important:** The game uses the `msvcrt` module for keyboard input, so it only works on Windows.

## How to run

Open your command prompt or PowerShell and type:

```bash
git clone https://github.com/MatrixAIPy/snake-python.git
cd snake-python
python Snake.py
```

## Controls:

* W - Up
* S - Down
* A - Left
* D - Right
